# Big Data Analytics and Machine Learning Fare Prediction NYC Taxi Data

This repository contains the Final Project (UAS) for the Big Data Analytics course. The project focuses on predicting New York City taxi fares based on trip distance and other key features using scalable data processing frameworks and machine learning models.

---

## 📌 Project Overview
Predicting taxi fares accurately helps both riders and fleet operators estimate costs efficiently. Using the massive NYC Taxi & Limousine Commission (TLC) Dataset, this project applies Big Data techniques to clean, process, and perform exploratory data analysis (EDA), followed by training predictive machine learning models.

### Key Objectives:
* Process large-scale taxi trip records efficiently.
* Perform Exploratory Data Analysis (EDA) to uncover distribution patterns and correlations.
* Develop and evaluate regression models to predict total fare amounts accurately based on trip distance.

---

## 🛠️ Tech Stack & Architecture
* **Language:** Python 3
* **Big Data Engine:** Apache Spark (`pyspark`)
* **Machine Learning:** `scikit-learn` / PySpark MLlib
* **Environment:** Jupyter Notebook
